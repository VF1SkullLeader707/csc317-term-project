About OrbitCart

OrbitCart is a CSC 317 full-stack course project at San Francisco State University. It simulates an aerospace e-commerce experience for learning and demonstration.

Project Overview

OrbitCart is a mock aerospace parts marketplace where users can browse products, manage a cart, and complete a simulated checkout flow. The platform is designed to demonstrate full-stack web development concepts including routing, sessions, database integration, and REST-style APIs.

Technology Stack

Backend: Node.js + Express
Database: SQLite
Frontend: Vanilla HTML, CSS, and JavaScript

Authentication and user state are handled through server-side sessions, and core product/cart functionality is implemented through API routes.

Academic Note

OrbitCart is a course project for CSC 317 at San Francisco State University. All products, users, orders, and checkout actions are simulated for educational purposes only. No real payments are processed and no real financial data is collected.
